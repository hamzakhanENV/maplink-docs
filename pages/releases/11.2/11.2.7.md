## Improvements
- **1843 Upgrade Earth SDK**
    > *This is a general release to replace 11.2.6 that provides this SDK for both Windows and Linux.*
    - Upgraded Earth SDK for MapLink Pro 11.
    - Updated openscenegraph to 3.6.5-20250604.1.
    - Updated osgearth to osgearth:3.7.3-20250604.1.
    
- **1852 Upgrade WMS Server**
    > *This is a general release to replace 11.2.6 that provides this SDK for both Windows and Linux.*
    - Upgraded WMS Server for MapLink Pro 11
    - Update base image to rocky 8
    - Update to tomcat v11
    - update to jakarta servlet 6.1
    - Update to java 25
